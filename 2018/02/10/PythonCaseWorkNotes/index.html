load() : Load a font file. This function loads a font object from the given bitmap font file, and returns the corresponding font object.

truetype() : Load a TrueType or OpenType font from a file or file-like object, and create a font object. This function loads a font object from the given file or file-like object, and creates a font object for a font of the given size.
